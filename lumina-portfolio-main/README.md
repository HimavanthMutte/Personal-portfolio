## üöÄ Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- MongoDB instance (local or Atlas)

### Installation

1. **Clone the repository**:
   ```sh
   git clone https://github.com/HimavanthMutte/lumina-portfolio.git
   cd lumina-portfolio
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Environment Setup**:
   Create a `.env` file in the root directory:
   ```env
   MONGODB_URI=your_mongodb_uri
   PORT=5000
   ```

4. **Seed the Database**:
   ```sh
   npm run seed
   ```

5. **Run the application**:
   ```sh
   npm run dev
   ```
